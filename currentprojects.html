Calculator project

In this project, our goal is to develop a scripting calculator but embedded on a (NXP) microcontroller. Therefore, we use a C library called Calc that can be integrated in a microcontroller and was developed to perform basic arithmetic operations. The project goes like this. We have a the PC that can communicate with the NXP board over UART. From the PC, we open a serial terminal and input math operation commands. These are received by the microcontroller which interprets them (Calc library). From there, it computes and sends back the result of the input operation over UART that ends up printed out on our serial monitor. We aim at developing further this calculator by adding new features to the system.

  1. Load files directly from an external storage device
  2. Display graphs on a LCD screen
  3. Use the two-core feature to split up tasks: one core that handles calculations and the other one that handles display.
  4. Integrate more advanced math operations such as FFT or matrix operations with PowerQuad.

The NXP microcontroller we use (LPCS55S69-EVK), is a two-core microcontroller.
